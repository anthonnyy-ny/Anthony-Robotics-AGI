### 题目解析

此问题涉及**平行四边形的性质**，已知平行四边形的三个顶点以及第四点和其中一个顶点重合，要求计算第四个点的坐标。

---
### 关键性质

- **平行四边形对角线性质**：
    - 在平行四边形中，两条对角线的中点重合。
    - 公式表示： A(x1,y1)+C(x3,y3)=B(x2,y2)+D(x4,y4) 如果已知三个顶点，可以通过这个公式计算第四个点。

---
### 解题步骤

1. **判断重复点**：
    
    - 在输入的四个点 `(x1, y1), (x2, y2), (x3, y3), (x4, y4)` 中，确定哪两个点是重合的。
    - 例如，若 `(x1, y1) == (x3, y3)`，说明点 `1` 和点 `3` 是同一点。
2. **计算第四点**：
    
    - 根据性质，假设重复的点是 `A` 和 `C`，那么： D(x,y)=(B(x2,y2)+D(x4,y4))−A(x1,y1)
    - 使用公式计算第四点的坐标。
3. **输出结果**：
    
    - 保留小数点后三位，格式化输出。
2nd
```cpp=
#include<bits/stdc++.h>
using namespace std;
int main(){
	double x1,y1,x2,y2,x3,y3,x4,y4,x,y;
	while(cin>>x1>>y1>>x2>>y2>>x3>>y3>>x4>>y4){
		//(1,3)
		if(x1==x3 && y1==y3){
			x=(x2+x4)-x1;
			y=(y2+y4)-y1;
		}//(1,4)
		else if(x1==x4 && y1==y4){
			x=(x2+x3)-x1;
			y=(y2+y3)-y1;
		}//(2,3)
		else if(x2==x3 && y2==y3){
			x=(x1+x4)-x2;
			y=(y1+y4)-y2;
		}//(2,4)
		else{
			x=(x1+x3)-x2;
			y=(y1+y3)-y2;
		}
		//output
		cout<<fixed<<setprecision(3)<<x<<" "<<y<<endl;
	}
	
	return 0;
}
```
tips- if判断跟着(1,3)(1,4)(2,3)(2,4) 计算用 先x2,x4(1,3)两数相加后减x1/y1或x2/y2
