#### list
---
1- list()
2- 可empty list []
3- 可以混合存储不同类型（int、str、float、对象等）
4- 可以multiple list
5- 可以append(),insert(),pop()
6- 可以修改内部参数不同类型(int、str、float、对象等)

```python=
example_list = [1, "a", 3.14]
example_list2 = [1, 2, 3, 4]
example_list3 = [1, "Up", ["Down", "Twice"]]
empty_list = []

example_list4 = [4, 5, 6]
example_list4.append(7)
example_list4 = [4, 5, 6,7]

example_list5 = [4, 5, 6, 7]
example_list5[1] = 'oh'
example_list5 = [4, 'oh', 6, 7]

example_list6 = [4, 5, 6, 7]
example_list6.insert(2, 5.5)
print(example_list6) # [4, 5, 5.5, 6, 7]

fruits_list = ["cherry", "lemon", "tomato", "apple", "orange"]
fruits_list.pop(2)
print(fruits_list) # ["cherry", "lemon", "apple", "orange"]
```
